Understanding Angular Pipes

Date Pipe Example

This demonstrates how to format dates using Angular's date pipe.

Current Date:

Date: {{ dateToday }}

Formatted Date (Uppercase):

Date Pipe: {{ dateToday | date | uppercase }}

Formatted Date (Short Format, Lowercase):

Date Pipe: {{ dateToday | date : "short" | lowercase }}

Name Pipe Example

This demonstrates how to manipulate string data using Angular's pipes.

Name:

Name: {{ name | uppercase }}

Slice Pipe:

Name (Sliced): {{ name | slice : 6 }}

Number Pipe Example

This demonstrates how to format numbers using Angular's number pipe.

Positive Number:

{{ 3.6 | number : "1.0-0" }}

Negative Number:

{{ -3.6 | number : "1.0-0" }}
