JavaScript String Methods

1. slice()

The slice() method extracts a section of a string and returns it as a new string.

Example:

const str = "Hello, World!";
const sliced = str.slice(7, 12); // "World"
console.log(sliced);

Output: World

2. toUpperCase()

The toUpperCase() method converts a string to uppercase letters.

Example:

const str = "hello";
const upper = str.toUpperCase(); // "HELLO"
console.log(upper);

Output: HELLO

3. toLowerCase()

The toLowerCase() method converts a string to lowercase letters.

Example:

const str = "HELLO";
const lower = str.toLowerCase(); // "hello"
console.log(lower);

Output: hello

4. concat()

The concat() method joins two or more strings and returns a new string.

Example:

const str1 = "Hello";
const str2 = "World";
const result = str1.concat(", ", str2); // "Hello, World"
console.log(result);

Output: Hello, World

5. indexOf()

The indexOf() method returns the first index at which a given element can be found in the string, or -1 if it is not present.

Example:

const str = "Hello, World!";
const index = str.indexOf("World"); // 7
console.log(index);

Output: 7

6. replace()

The replace() method returns a new string with some or all matches of a pattern replaced by a replacement.

Example:

const str = "Hello, World!";
const newStr = str.replace("World", "JavaScript"); // "Hello, JavaScript!"
console.log(newStr);

Output: Hello, JavaScript!
